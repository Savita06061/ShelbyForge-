SHELBYFORGE
Forge Your Verifiable Storage • Powered by Shelby + AptosShelbyForge is a decentralized verifiable hot vault that enables users to securely upload files, generate cryptographic hashes, and register proof references using blockchain infrastructure. Overview
ShelbyForge allows users to:Upload files securely
Generate cryptographic hashes (client-side)
Interact with Aptos wallet (Petra)
Trigger transaction flow for proof registration
Store files using Shelby’s decentralized storage
The goal is to create a simple system where digital files can be secured and later verified. How It Works
User connects Petra wallet
File is uploaded through the UI
A cryptographic hash is generated in the browser
Transaction flow is triggered (wallet interaction)
File is sent to Shelby storage
Reference data can be used later for verification
 Key Features
Client-side cryptographic hashing
Wallet-based authentication (Petra)
Transaction interaction on Aptos Testnet
Shelby decentralized storage integration
Simple and clean dashboard UI
