# 🚀 ShelbyForge

> Secure • Verifiable • Decentralized Hot Vault  
> Built by the **ShelbyForge Team**

ShelbyForge is a premium personal secure vault that enables users to upload files, generate cryptographic proofs, and register file integrity on the Aptos Testnet using Shelby’s verifiable storage infrastructure.

Designed for transparency, ownership, and trust — ShelbyForge combines decentralized storage with blockchain-based verification to create a modern verifiable vault experience.

🌐 Live Demo: https://shelbyforge.vercel.app

---

## ✨ Features

- 🔐 Secure file upload system
- 🧾 Cryptographic proof generation
- ⛓️ Aptos Testnet proof registration
- 📦 Shelby verifiable decentralized storage
- ⚡ Fast and minimal user experience
- 🛡️ File integrity verification
- 🌍 Transparent and tamper-resistant architecture

---

## 🧠 How ShelbyForge Works

```text
User Uploads File
        ↓
Generate Cryptographic Hash
        ↓
Store File on Shelby
        ↓
Register Proof on Aptos Testnet
        ↓
Verify File Integrity Anytime
```

ShelbyForge ensures that uploaded files remain verifiable, traceable, and protected through decentralized infrastructure.

## ⚙️ Local Development

```bash
# Clone repository
git clone https://github.com/Savita06061/ShelbyForge-.git

# Enter project directory
cd ShelbyForge-

# Install dependencies
npm install

# Start development server
npm run dev
```

## 🔑 Environment Variables

Create a `.env.local` file:
