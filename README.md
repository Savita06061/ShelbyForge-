# 🔐 ShelbyForge – Verifiable Decentralized Hot Vault

ShelbyForge is a **fully decentralized, verifiable hot vault** built on **Aptos & Shelby Protocol**, enabling users to securely store files, generate cryptographic proofs, and verify data integrity on-chain.

🌐 **Live Demo:** https://shelbyforge.vercel.app/

## 🧠 Overview

ShelbyForge allows users to:

* 📁 Securely upload files
* 🔐 Generate cryptographic hashes client-side
* 👛 Connect Petra Wallet
* ⛓️ Register proof references on Aptos Testnet
* ☁️ Store files on Shelby decentralized hot storage

🎯 **Goal:**
Make digital files **tamper-proof, verifiable, and trustless** using blockchain technology.

---

## ⚙️ How It Works

1. 👛 Connect Petra Wallet
2. 📁 Upload file via UI
3. 🔐 Hash generated in browser (client-side)
4. ⛓️ Transaction executed on Aptos Testnet
5. ☁️ File stored on Shelby decentralized storage
6. ✔️ Proof recorded & verifiable on-chain
