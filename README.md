# SHELBYFORGE

**Forge Your Verifiable Storage** • Powered by **Shelby + Aptos**

![ShelbyForge Banner](https://via.placeholder.com/800x200/0A2540/00D4FF?text=SHELBYFORGE) <!-- Replace with actual banner later -->

ShelbyForge is a decentralized **verifiable hot vault** that lets users securely upload files, generate cryptographic proofs client-side, and register immutable references on the Aptos blockchain.

---

## 🧠 Overview

ShelbyForge enables anyone to:
- Upload files securely
- Generate cryptographic hashes directly in the browser
- Register proof references on Aptos using Petra wallet
- Store files on Shelby’s decentralized hot storage

This creates a transparent and verifiable system for digital assets — perfect for content provenance, AI data, legal records, and personal archives.

---

## ⚙️ How It Works

1. Connect your **Petra Wallet**
2. Upload a file through the intuitive UI
3. Cryptographic hash is generated **client-side**
4. Trigger a transaction to register the proof on Aptos Testnet
5. File is stored on **Shelby decentralized storage**
6. Get a verifiable reference for future checks

---

## 🔐 Key Features

- **Client-side hashing** – Your files never leak during proof generation
